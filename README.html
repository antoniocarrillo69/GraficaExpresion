Jerarquía de clases que permite solicitar una cadena que contenga una expresión del tipo:
    y=sinl(x)*x
y el programa genera la gráfica mostrandola en formato texto

Los fuentes de este proyecto puede descargarse de GitHub mediante
    $ git clone git://github.com/antoniocarrillo69/GraficaExpresion.git
Para compilar usar:
    $ g++ *.cpp
Ejecutar mediante:
    $ ./a.out
